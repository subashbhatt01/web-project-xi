About AdobePhotoshop

Adobe Photoshop is a photo editing software developed by Adobe Systems. It is widely used by photographers, graphic designers, and artists for image editing, retouching, and manipulation.

The software has a wide range of features including color correction, image resizing, cropping, layering, masking, and compositing. It also supports various file formats such as JPEG, PNG, GIF, BMP, TIFF, and PSD.

Adobe Photoshop is available for purchase through Adobe's Creative Cloud subscription service. The subscription includes access to other Adobe products such as Illustrator, InDesign, Premiere Pro, and more.

Some keyboard shortcuts for AdobePhotoshop

  • Undo a single action within your project: Ctrl + Z (Windows) Cmd + Z (macOS)
  • Undo multiple actions within your project: Ctrl + Alt + Z (Windows) Cmd + Z repeatedly (macOS)
  • Access the Zoom tool by using the Photoshop Zoom shortcut: Z
  • Change image size: Control + Alt + i (Command + Option + i )
  • Change canvas size: Control + Alt + c (Command + Option + c )
  • Zoom out: Control + - (Command + - )
  • Zoom in: Control + + (Command + + )
